# Fee & Elfe — Website UI Kit

## Overview
Marketing website prototype for Fee & Elfe Consulting. A single-page scrollable site covering all primary sections.

## Screens / Sections
| Section | Description |
|---|---|
| Navigation | Sticky header with logo, links, CTA button |
| Hero | Full-bleed dark pine hero with display typography and stats |
| Services | 3-column service cards grid (default, featured dark, accent) |
| Team | Fee & Elfe consultant profiles |
| Testimonials | 3-column client quotes grid |
| CTA Banner | Full-width conversion section |
| Footer | 4-column footer with links |

## Design Width
1280px — desktop-first, fluid layout using max-width container.

## Fonts
- Cormorant Garamond (Google Fonts) — display/headings
- DM Sans (Google Fonts) — body/UI

## Usage
Open `index.html` directly. All assets reference `../../assets/` (relative to ui_kits/website/).
